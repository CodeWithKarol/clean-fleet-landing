## Asset Guidelines

### Image Specifications

- **Format**: JPG for photos, PNG for graphics with transparency, SVG for logos
- **Quality**: High-resolution (minimum 2x for retina displays)
- **Optimization**: Compressed for web while maintaining quality
- **Alt Text**: All images should have descriptive alt text for accessibility

### Video Specifications

- **Format**: MP4 with H.264 encoding
- **Duration**: 30-60 seconds
- **Resolution**: 1920x1080 (Full HD)
- **Audio**: Optional background music, ensure accessibility compliance

### Brand Colors

- **Primary Green**: #2d6e3e
- **Accent Green**: #4ade80
- **Dark Gray**: #1a202c
- **Light Gray**: #f8fafc
- **White**: #ffffff
